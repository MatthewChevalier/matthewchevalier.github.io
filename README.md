# **Personal WebSite**
## *📝 Descrizione del Progetto*
Sito web personale sviluppato per mostrare il proprio portfolio e creare una vetrina online delle proprie competenze, progetti e contatti.

## *💻 Tecnologie Utilizzate*
- HTML
- CSS
- JavaScript

## *🚀 Funzionalità*
-Visualizzazione dei progetti del portfolio con descrizione e immagini.
-Sezione “Chi sono” per presentarsi professionalmente.
-Modulo di contatto funzionante.
